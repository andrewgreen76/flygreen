
===================================================================================
DNS SERVERS 
===================================================================================

DNS server shapes : 
 . a hardware appliance 
 . general-purpose computer 
 . a virtual machine running DNS software 


DNS server basics : 
 . a.k.a. name server
 . takes the address as a text string , looks up the IP address , and provides it to the client 


RESOLVER and DNS QUERIES : 
 . client-side software 
 . mediates between the client and the DNS server
