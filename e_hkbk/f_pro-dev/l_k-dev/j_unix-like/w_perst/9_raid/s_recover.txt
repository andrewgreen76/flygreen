================================================================================================================================
DATA RECOVERY 
================================================================================================================================

 . once the faulty disk is identified, it is replaced, but the data on it is gone 
 + XOR is used to calculate the parity bit for every stripe 
 + XOR is also used to recover lost data
   . XOR all the good data bits and the parity bit on the same stripe
   = your lost data bit
 . checking and correction are implemented together as ECC (error-correction code) 



================================================================================================================================
	Error detection 
================================================================================================================================

               [disk 0:]   [disk 1:]   [disk 2:]  [disk 3:]   
   stripe A        1    (+)    0    (+)    1    (+)   0	      = 0  <- your "old" parity bit (O) - stored on the parity disk 
   .............................................................................................................................
   stripe A	   1    (+)    1    (+)    1    (+)   0	      = 1  <- your new parity bit (N) - doesn't match the old parity bit 
   	  	   	       	    	   	      	      	      	       	      	      	=> we have an error



================================================================================================================================
	Error correction 
================================================================================================================================

               [disk 0:]   [disk 1:]   [disk 2:]  [disk 3:] [parity disk]  
   stripe A	   1           x           1          0	        0 
   .............................................................................................................................
   stripe A	   1    (+) - - - - - - -  1    (+)   0	 (+)    0		= 0 = x
   .............................................................................................................................
   stripe A        1    (+)    0    (+)    1    (+)   0	        0
